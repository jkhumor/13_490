<h3>课程介绍：</h3>
<p>计算机软件界较大的变迁是从串行编程转向了并行编程。其中，CUDA起到了重要的作用。究其本质，图形处理单元（Graphics Processor Unit，<a title="查看与 GPU 相关的文章" target="_blank">GPU</a>）是为高速图形处理而设计的，它具有天然的并行性。CUDA采用了一种简单的数据并行模型，再结合编程模型，从而无需操纵复杂的图形基元。我们也可以轻而易举的发现各种各样的开源计算机库，例如<a class="relatedlink" target="_blank" rel="noopener">OpenCV</a>,Caffe,TensorFlow等等，都提供了对应于<a title="查看与 GPU 相关的文章" target="_blank">GPU</a>的使用接口，使得GPU的使用越来越主流而高效。</p>

<h3>适用人群：</h3>
<p>对并行计算，GPU编程刚兴趣的同学，熟悉C/C++编程语言。需要有带NVIDIA显卡的计算机，如果没有的同学可能需要额外购买AWS的GPU服务器，费用约是0.4美元/小时。</p>
